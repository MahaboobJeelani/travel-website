# Project Overview

**Roamify** is a premium, enterprise-grade travel platform built with modern web technologies. It addresses the common problem of fragmented travel planning by providing a unified platform where users can discover hotels worldwide, save favorites, and access authentic travel blogs - all within a single, seamless application.

## Live Demo

 **Live Website:** [https://travel-website-v6un.onrender.com](https://Roamify.vercel.app)  
 **Mobile Responsive:** Yes  
 **Performance:** 90+ Lighthouse Score  

## Features

### Core Features
-  **Hotel Discovery**: Country-based search with detailed hotel information
-  **Favorites System**: Persistent favorites with local storage
-  **Travel Blogs**: Dynamic blog content with detailed articles
-  **Responsive Design**: Mobile-first approach for all devices
-  **Premium UI**: Modern, enterprise-grade design aesthetic

### Hotel Management
- **Country-based Search**: Browse hotels from 15+ countries
- **Detailed Hotel Pages**: Image carousels, amenities, room types, reviews
- **Real-time Pricing**: Dynamic price display with currency formatting
- **Amenity Mapping**: Intelligent icon mapping for hotel amenities
- **Room Selection**: Multiple room types with detailed descriptions

### User Experience
- **Persistent Favorites**: Hotels saved across browser sessions
- **Interactive Notifications**: Real-time feedback with react-hot-toast
- **Smooth Animations**: Fade-in, slide-up, and hover effects
- **Intuitive Navigation**: Clean routing with React Router DOM
- **Loading States**: Skeleton loaders and progress indicators

### Content Management
- **Dynamic Blogs**: Paginated blog listings with detailed articles
- **Related Content**: Smart related article recommendations
- **Category Filtering**: Organized content by travel categories
- **Author Profiles**: Detailed author information and bios

## Tech Stack

### Frontend
- **React 18.2.0**: Functional components with hooks
- **Redux 4.2.1**: State management with Redux Thunk middleware
- **React Router DOM 6.16.0**: Client-side routing
- **Tailwind CSS 3.3.0**: Utility-first CSS framework
- **React Icons 4.11.0**: Comprehensive icon library
- **Axios 1.5.1**: HTTP client for API requests

### State Management
- **Redux**: Predictable state container
- **Redux Thunk**: Middleware for async operations
- **Local Storage**: Client-side data persistence
- **Context API**: Component-level state management

### UI/UX Libraries
- **react-hot-toast**: Beautiful notification system
- **Tailwind CSS Forms**: Form styling plugin
- **Tailwind CSS Typography**: Blog content styling

## Installation Process

- cd Roamify

### 2. Install dependencies
- npm install

### 3. Start development server
- npm start
 - or
